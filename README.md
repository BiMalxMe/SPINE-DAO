# 🧠 SpineAI CLI – Rust Prototype for SpineDAO

This is a Rust CLI prototype simulating data preprocessing for SpineDAO’s AI stack. It’s designed to help with lumbar spine MRI dataset preparation, normalization, and integration with future AI tools for back pain prevention.

---

## 📦 Features
- Loads configuration from `.env` (simulates API keys, dataset paths)
- Simulates preprocessing of lumbar MRI dataset
- CLI output showing step-by-step pipeline stages
